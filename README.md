# Strime
A simple string (time+unit) to seconds converter.


## Installation
> Linux:
> ```
>  python3.10 -m pip install regfex
> ```
> Windows:
> ```
>  pip install regfex
> ```

## Using
> ```python
> import strime
>
> example = strime.Strime("5d") # 5d = 5 days.
> print(example.seconds)
> ```
> Output:
> ```
> 432000
> ```
